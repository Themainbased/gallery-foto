# GALLERY FOTO

Gallery foto sebuah platform sosial media berbasis gambar

## Bahan

- PHP v 8.2.0
- Bootstrap
- Web Browser ( Microsoft Edge )


## Fitur

- Unique ID ( Foto dengan numberic unik )
- Encryption ( data password user )
- Profile ( Edit data pribadi user )
- Like
- Komentar
- Album
- Upsize Foto Size
